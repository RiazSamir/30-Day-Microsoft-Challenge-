# 30-Day-Microsoft-Challenge

This is 30 day hands on exploration of real SOC operations using the Microsoft Security stack. This project documents KQL investigations, incident response procedures, and key lessons learned while developing modern cloud-focused detection capabilites. 

# Overview 

This Projects focuses on building a cloud based SOC enviroment utilisng Microsofts Security Stack. This includes:

- Microsoft Sentinel
- Microsoft Defender XDR (Extended Detection & Response)
- Microsoft Defender for Endpoint
