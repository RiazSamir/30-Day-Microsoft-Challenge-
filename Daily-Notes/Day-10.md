# Day 10 - Creating Test Users

## Objective: 
- Create test users for later phishing simulations
- Familiarise with Email security Investigation features in Microsoft Defender

## Platform / Tools
- Microsoft 365 Admin Centre
- Microsoft Entra ID
- Microsoft Defender XDR

## Actions Performed 
- Created 2 test users via the Microsoft 365 Admin Centre (*Figure 1*)
- Assigned Microsoft 365 E5 licenses to enable access to products such as Outlook
- Activated the Mailboxes for both users via Outlook Web Access
  - Explored Email & Collaboration security features, including (*Figure 2*):
  - Explorer (email and URL click investigations)
  - Review (user-reported and quarantined messages)
  - Campaigns (phishing campaign grouping)
  - Threat Tracker (global threat insights)
  - Exchange Message Trace
  - Attack Simulation Training
